# CV Web - Nazwa Syafiya Mirzal

Proyek ini merupakan CV Web yang dibuat menggunakan HTML, CSS, dan PHP.  
Aplikasi ini menampilkan informasi pribadi, pengalaman, keahlian, dan portofolio secara menarik dan responsif.

## Fitur Utama
- Halaman Utama (Home): menampilkan profil singkat dan deskripsi diri.
- Pengalaman & Pendidikan: berisi daftar riwayat pendidikan dan pengalaman.
- Portofolio: menampilkan proyek dalam bentuk link menuju karya Canva.
- Desain Responsif: tampilan menyesuaikan di perangkat desktop maupun mobile.

## Teknologi yang Digunakan
- Frontend: HTML5, Tailwind CSS, JavaScript
- Backend: PHP (CodeIgniter 4)
- Database: MySQL
- Tools: Laragon, GitHub
